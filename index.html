WHAT IS ECMASCRIPT 6?

What is ECMAScript?

Official name of the programming language commonly known as JavaScript

Specification ECMA-262 maintained by Ecma International's technical committee TC-39

WHY IS ES6 SUCH A BIG DEAL?

ECMAScript, 5th Edition (a.k.a. ES5)

  • Approved as a standard in December 2009
  • Scaled back subset of abandoned 4th edition focused mainly on updating existing features

ESHarmony

Versions of abandoned features from ES4

ES.next => ES6

Subset of features from ESHarmony which would eventually become known as ECMAScript, 6th Edition

"Subset"

Most significant update to the core of the language since the introduction of [] and {} in JavaScript 1.2!

Reduced friction

Incorporates concepts common amongst other popular languages with the goal of making it easier to get up and running for developers used to those languages

Destructuring assignment

assignment from an array


// es5
var data = [10, 10, 50, 200];

var x = data[0];
var y = data[1];
var width = data[2];
var height = data[3];
						

// es6
let data = [10, 10, 50, 200];
let [x, y, width, height] = data;

assignment from an object


// es5
var data = {name: 'John Doe', age: 36};
var name = data.name;
var age = data.age;
						

// es6
let data = {name: 'John Doe', age: 36};
let {name, age} = data;
